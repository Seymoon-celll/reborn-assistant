🎨 Interface & Personnalisation

Personnalisez Reborn Assistant selon vos préférences

Thèmes visuels

Reborn Assistant propose 6 thèmes pour personnaliser l'apparence de l'interface :

MMO Gold
Cyber Blue
Neon Purple
Fire Red
Dark Mode
White Mode

Changer de thème

1. Ouvrez le panneau Reborn
2. Cliquez sur l'onglet "⚙️ CONFIGURATION"
3. Sélectionnez votre thème dans le menu déroulant
4. Le changement est instantané

4. Toute l'interface se traduit instantanément

Raccourcis clavier

Reborn Assistant inclut des raccourcis globaux pour un contrôle rapide :

Raccourcis système

  • CTRL + A : Démarrer le bot
  • CTRL + O : Arrêter le bot

Raccourcis personnalisables

Vous pouvez configurer les touches pour :

  • Actions de type 1 et 2
  • Macros (séquences de touches)
  • Macro périodique

Options d'affichage

Position du panneau

Le panneau Reborn est déplaçable :

  • Cliquez sur l'en-tête du panneau
  • Maintenez le bouton enfoncé et déplacez
  • Relâchez à l'endroit souhaité
  • La position est sauvegardée automatiquement

Opacité du panneau

Ajustez la transparence pour ne pas gêner votre navigation tout en gardant un œil sur vos statistiques.

Log d'activité

Le log affiche en temps réel toutes les actions effectuées :

  • Activation/désactivation des systèmes
  • Utilisation de potions
  • Exécution des macros et séquences
  • Appels de la macro périodique
  • Erreurs éventuelles

Sauvegarde des paramètres

Toutes vos configurations sont automatiquement sauvegardées localement :

  • Thème sélectionné
  • Langue de l'interface
  • Seuils déclencheurs
  • Touches assignées
  • Configuration des macros
  • Position du panneau

Elles persistent entre les sessions, même après fermeture du navigateur.
