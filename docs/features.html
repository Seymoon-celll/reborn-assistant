Fonctionnalités GRATUITES

💊
GRATUIT
Action automatique de type 1
Déclenchez automatiquement votre action de type 1 (HP) selon un timer configurable.
  • Timer d'action configurable
  • Fonctionne en arrière-plan
🎨
GRATUIT
Thèmes visuels
Personnalisez l'apparence de l'interface avec 6 thèmes différents.
  • MMO Gold (par défaut)
  • Cyber Blue, Neon Purple, Fire Red
  • Dark Mode & White Mode
🌍
GRATUIT
Multi-langues
Interface disponible en 14 langues pour une expérience internationale.
  • Français, English, Español, Deutsch, Italiano, Português, Nederlands
  • Polski, Русский, 中文, 日本語, 한국어, Filipino, Türkçe
  • Changement instantané

Fonctionnalités PREMIUM 💎

❤️
PREMIUM
Macro déclencheur 1 automatique
Déclenchement automatique de votre action de type 1 (HP) selon un timer configurable.
  • Timer déclencheur 1 configurable
  • Touche de raccourci configurable
  • Fonctionne en arrière-plan
💧
PREMIUM
Macro déclencheur 2 automatique
Utilisation automatique de vos actions de type 2 selon un timer configurable.
  • Timer déclencheur 2 configurable
  • Touche de raccourci configurable
  • Fonctionne en arrière-plan
⌨️
PREMIUM
Raccourcis clavier
Contrôlez Reborn Assistant à la volée avec des raccourcis globaux personnalisables.
  • CTRL+A : Démarrer le bot
  • CTRL+O : Arrêter le bot
🛡️
PREMIUM
Rotation de séquences
Système automatique de maintien des séquences avec une séquence prédéfinie.
  • Séquence : F2 → 1-9 → 0 → F1
  • Intervalle configurable (minutes)
  • Délai entre touches réglable
⚙️
PREMIUM
Macros Personnalisables
Créez jusqu'à 3 macros avec vos propres séquences de touches.
  • 3 macros indépendantes
  • Séquences personnalisées
  • Intervalles de répétition réglables
🐉
PREMIUM
Macro périodique
Appel et stockage automatique de votre macro périodique toutes les 15 secondes.
  • Appel automatique de la macro périodique
  • Auto-stockage après appel
  • Intervalle fixe de 15 secondes
