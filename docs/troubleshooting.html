Solutions aux problèmes courants et FAQ

Problèmes d'installation

❌ L'extension n'apparaît pas dans la barre d'outils
Solutions :
1. Vérifiez que l'extension est bien installée depuis le Chrome Web Store
2. Épinglez l'extension : cliquez sur l'icône puzzle 🧩 dans la barre d'outils et épinglez Reborn Assistant
3. Redémarrez votre navigateur
4. Si le problème persiste, désinstallez puis réinstallez l'extension depuis le Chrome Web Store

Problèmes de connexion

❌ "Connexion échouée" ou "Licence invalide"
Causes possibles :
• Mauvaise connexion internet
• Serveurs Google Cloud temporairement indisponibles
• Licence expirée ou révoquée

Solutions :
1. Vérifiez votre connexion internet
2. Déconnectez-vous et reconnectez-vous avec Google
3. Vérifiez votre statut de licence dans votre espace Stripe
4. Contactez le support si le problème persiste
❌ Le panneau indique "FREE" alors que j'ai Premium
1. Déconnectez-vous du compte Google
2. Fermez complètement le navigateur
3. Rouvrez et reconnectez-vous
4. Si le problème persiste, vérifiez que vous êtes connecté avec le bon compte Google (celui utilisé pour l'achat)

Problèmes de fonctionnement

❌ Les actions de type 1/2 ne se déclenchent pas
⚠️ IMPORTANT
Les actions de type 1 et 2 sont des fonctionnalités PREMIUM UNIQUEMENT. Seule l'action automatique de type 1 (HP) est disponible en version gratuite.
Si vous avez Premium :
1. Le bot est-il démarré ? CTRL+A
2. Les toggles déclencheurs sont-ils activés ? (indicateur vert « ON »)
3. Les touches configurées sont-elles correctes ?
❌ Les séquences / macros / macro périodique / raccourcis ne fonctionnent pas
⚠️ IMPORTANT
Les raccourcis clavier, séquences, macros et macro périodique sont des fonctionnalités PREMIUM UNIQUEMENT. En version gratuite, seule l'action automatique de type 1 (HP) est disponible.
Si vous avez Premium :
1. Vérifiez que votre licence Premium est active (icône 💎 dans le panneau)
2. Les toggles correspondants sont-ils activés ?
3. Vérifiez les intervalles et délais configurés
❌ Le bot s'arrête tout seul
Causes possibles :
• Perte de connexion à Google Cloud
• Détection DevTools
• La page a changé d'URL

Solutions :
1. Redémarrez le bot avec CTRL+A (Premium) ou via le panneau
2. Rechargez la page concernée
3. Vérifiez votre connexion internet
4. Ne pas ouvrir les DevTools si vous êtes en Premium

Optimisation & Performance

💡 Le bot consomme trop de ressources
Recommandations :
• Augmentez les intervalles des macros et séquences (moins d'actions = moins de CPU)
• Désactivez les systèmes dont vous n'avez pas besoin
• Fermez les autres onglets inutiles
• Utilisez un navigateur optimisé (Brave, Edge sont plus légers que Chrome)
💡 Comment optimiser les paramètres ?
Configuration recommandée :
• Intervalle séquences : 5-25 minutes (selon durée de vos séquences)
• Délai entre touches : 1-2 secondes (évite les erreurs)
• Macros : Intervalles de 10-30s selon l'usage

FAQ Générale

❓ Puis-je utiliser Reborn sur plusieurs navigateurs ?
Oui, mais vous devez installer l'extension sur chaque navigateur. Votre licence Premium fonctionne sur tous vos appareils tant que vous vous connectez avec le même compte Google (1 connexion simultanée max).
❓ Mes données sont-elles sauvegardées ?
Toutes vos configurations (touches, macros, thème, langue) sont sauvegardées localement dans le navigateur. Elles persistent entre les sessions. En revanche, si vous désinstallez l'extension, vous perdez ces données.
❓ Reborn peut-il provoquer un bannissement ?
L'utilisation d'outils d'automatisation peut violer les CGU de certains services. Vous êtes seul responsable de l'utilisation que vous faites de Reborn. Consultez les conditions d'utilisation du service concerné avant d'utiliser cet outil.
❓ Comment annuler mon abonnement Premium ?
Vous pouvez annuler à tout moment depuis votre portail de gestion Stripe (lien reçu par email après le paiement). L'annulation prend effet à la fin de votre période en cours. Aucun frais d'annulation.
❓ Quid du plan À Vie si le service ferme ?
Le plan À Vie garantit l'accès tant que le service est actif. L'extension une fois installée fonctionne localement ; seule la vérification de licence nécessite une connexion internet.
