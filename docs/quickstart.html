🚀 Démarrage Rapide

Installez et configurez Reborn Assistant en quelques minutes

Prérequis

🌐
Navigateur compatible : Google Chrome, Microsoft Edge, Brave ou tout navigateur basé sur Chromium
🔧
Compte Google : Requis uniquement pour l'authentification Premium (optionnel pour version gratuite)
🎮
Site compatible : Compatible avec les sites web standards

Installation

1
Visiter le Chrome Web Store

Rendez-vous sur la page Reborn Assistant dans le Chrome Web Store et cliquez sur « Ajouter à Chrome ».

💡 Compatible avec Google Chrome, Microsoft Edge, Brave et tout navigateur basé sur Chromium.

2
Confirmer l'installation

Cliquez sur « Ajouter l'extension » dans la fenêtre de confirmation. L'extension s'installe automatiquement.

3
Épingler l'extension

Cliquez sur l'icône puzzle 🧩 dans la barre d'outils de votre navigateur, puis épinglez Reborn Assistant pour un accès rapide.

4
Connexion (optionnelle pour Premium)

• Cliquez sur l'icône Reborn dans la barre d'outils
• Pour Premium : Connectez-vous avec votre compte Google
• Pour Gratuit : Utilisez directement sans connexion

✅ Installation réussie ! L'icône Reborn devient verte lorsque l'extension est active. Vous pouvez maintenant configurer vos fonctionnalités.

Configuration initiale

Version GRATUITE

Fonctionnalités disponibles sans connexion :

• Action automatique de type 1 (HP uniquement)
• 6 thèmes visuels
• Support de 14 langues

Configuration de l'action automatique de type 1

1. Ouvrez le panneau Reborn (clic sur l'icône)
2. Réglez le timer de l'action HP (par défaut : 2s)
3. L'action se déclenche automatiquement
4. Aucune connexion requise

Version PREMIUM 💎

Fonctionnalités additionnelles (nécessite connexion Google) :

• Macro déclencheur 1 automatique (timer configurable + raccourci)
• Macro déclencheur 2 automatique (timer configurable + raccourci)
• Raccourcis clavier (CTRL+A / CTRL+O)
• Rotation automatique de séquences (F2 → 1-9 → 0 → F1)
• 3 macros personnalisables avec séquences de touches
• Système de familier périodique automatique (appel + stockage toutes les 15s)

Configuration Premium

Séquences automatiques :
• Activez « Séquence Rotation » dans le panneau
• Configurez l'intervalle entre les cycles (défaut : 5 min)
• Ajustez le délai entre chaque touche (défaut : 1s)

Macros :
• Activez Macro 1, 2 ou 3
• Définissez la séquence de touches (ex: 1,2,3,4)
• Configurez l'intervalle de répétition

Macro périodique :
• Activez le toggle « Macro périodique »
• Le système appelle automatiquement votre macro périodique toutes les 15s
• Stockage automatique après l'appel

⚠️ Note importante Les fonctionnalités Premium (Macros déclencheur 1 et 2, Séquences automatiques, Macro périodique automatique, 3 macros configurables) nécessitent une licence Premium valide et une connexion Google. Sans licence, seule la macro de base est accessible.

Raccourcis clavier

CTRL + A : Démarrer le bot
CTRL + O : Arrêter le bot
