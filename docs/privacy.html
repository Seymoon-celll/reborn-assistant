🔒 Politique de Confidentialité

Comment nous protégeons vos données personnelles

✅ Votre vie privée est notre priorité
Version gratuite : aucune donnée personnelle collectée. Version Premium : seules les données strictement nécessaires à la vérification de votre licence sont traitées.

Collecte de données

Version Gratuite

Aucune donnée personnelle n'est collectée ni transmise. Toutes vos préférences sont stockées uniquement sur votre appareil.

Version Premium

Pour vérifier votre licence et sécuriser votre compte, les données suivantes sont traitées :

  • Votre adresse email Google (via authentification Google)
  • Un identifiant d'appareil unique (généré localement)
  • Votre statut d'abonnement et horodatages de session
  • Maximum 2 appareils enregistrés par compte

Base légale : exécution du contrat (art. 6.1.b RGPD).

Données jamais collectées

  • ❌ Aucune donnée de gameplay ou d'activité
  • ❌ Aucun historique de navigation
  • ❌ Aucun cookie publicitaire ou de tracking

Données stockées localement

Les informations suivantes sont sauvegardées uniquement sur votre appareil :

  • Préférences de langue : Votre choix de langue d'interface
  • Sélection du thème visuel : Le thème choisi parmi les 6 disponibles
  • Raccourcis clavier configurés : Les touches assignées pour vos actions
  • Intervalles de timer personnalisés : Les timers configurés pour vos macros
  • Position de l'interface : L'emplacement du panneau Reborn sur votre écran
  • Cache de licence (Premium) : identifiant d'appareil et statut d'abonnement, pour éviter une vérification à chaque démarrage

Services tiers

Services utilisés (version Premium)

Pour l'authentification et les paiements, Reborn Assistant utilise :

  • 🔐 Google — Authentification et stockage sécurisé des données de licence
  • 💳 Stripe — Traitement des paiements Premium (vos données bancaires ne transitent jamais par nos serveurs)

Ces prestataires sont conformes au RGPD et disposent des garanties appropriées pour les transferts internationaux de données.

Aucun tracking, aucune publicité

  • ❌ Aucun service d'analyse ou de statistiques
  • ❌ Aucune publicité ni marketing ciblé
  • ❌ Aucun cookie tiers ou suivi comportemental

Authentification et gestion des appareils

Connexion Google (version Premium)

Votre compte Google est utilisé uniquement pour authentifier votre identité et vérifier votre licence. Votre appareil est enregistré dans notre système sécurisé (maximum 2 appareils par compte).

Vous recevez une notification si votre compte est utilisé sur un appareil non reconnu. Vous pouvez gérer vos appareils enregistrés directement depuis l'extension.

Paiements Sécurisés

Les paiements Premium sont traités par Stripe. Vos informations bancaires ne transitent jamais par nos serveurs. Nous ne stockons aucune donnée de carte bancaire.

Durée de conservation

Vos données sont conservées uniquement le temps nécessaire :

  • Compte actif : données conservées pour toute la durée de l'abonnement
  • Après résiliation : suppression sous 30 jours
  • Données locales : supprimées immédiatement à la désinstallation de l'extension

Vos droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants sur vos données :

  • ✅ Accès — Obtenir une copie de vos données personnelles
  • ✅ Rectification — Corriger des données inexactes
  • ✅ Effacement — Demander la suppression de votre compte et de vos données
  • ✅ Portabilité — Recevoir vos données dans un format lisible
  • ✅ Réclamation — Déposer une plainte auprès de la CNIL (cnil.fr)

Pour exercer ces droits, contactez-nous à : seymoon.cell@gmail.com ou sur Discord (Seymoon)

Suppression de vos données

1. Désinstallation de l'extension
En désinstallant Reborn Assistant, toutes les données locales sont automatiquement supprimées.

2. Réinitialisation manuelle
Vous pouvez réinitialiser les paramètres depuis l'interface de Reborn à tout moment.

3. Suppression du compte (Premium)
Pour supprimer votre compte Premium et toutes les données associées, contactez-nous par email.
